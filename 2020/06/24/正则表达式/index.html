正则表达式（构造器RegExp）

——JavaScript中的RegExp对象

正则：规律、规则

表达式：有一些普通字符和特殊字符组成

强大的字符串匹配工具

对字符串操作的一种逻辑公式

表达对字符串的一种过滤逻辑

正则表达式是匹配模式，要么匹配字符，要么匹配位置

正则表达式测试工具

https://regex101.com/

https://jex.im/regulex/

创建正则表达式

①、new 运算符 new RegExp(pattern, flags)
var box = new RegExp('box'); //第一个参数字符串
var box = new RegExp('box','ig'); //第二个参数可选模式修饰符
②、字面量方式 /pattern/ flags
var box = /box/; //直接用两个反斜杠
var box = /box/ig; //在第二个斜杠后面加上模式修饰符

修饰符：每个正则表达式都可带一个或多个标志（flags），标明正则表达式的行为

==模式修饰符的可选参数：==

参数 含义
i 忽略大小写
g 全局匹配
m 多行匹配

==转义字符：==

元字符是具有特殊含义的字符，不能直接匹配使用的，必须使用反斜线（\） 作为前缀进行转义后才能使用。

var pattern = new RegExp('\\\\'); //先解析为字符串，再解析为正则

一般不推荐使用构造函数生成正则，而应该优先使用字面量。
var pattern = new RegExp("^\\d+","g"); //注意，反斜杠需要转义
var pattern =/^\d+/g;
构造函数会需要多写一个
构造函数的匹配模式可以使用变量（可动态获取字符串），但是字面量不能够使用变量代替

正则方法

RegExp原型方法：基于原型对象 RegExp.prototype

  • test()：检索字符串中指定的值，返回true或false regObject.test(str);
  • exec()：检索字符串中指定的值，返回找到的值，并确定其位置 regObject.exec(str);

String原型方法（前三个一般与正则表达式搭配使用）

  • replace()：

    strObject.replace(regexp,'replaceString');

    对字符串中特定格式的子串进行替换，返回替换后的结果
    第一个参数既可以是一个固定的子串，也可以是一个正则表达式对象

  • match()

    strObject.match(regexp);
    一个或多个子串、正则表达式的匹配
    匹配成功返回一个数组，匹配不成功返回null；多个匹配返回匹配到的内容

  • search()

    strObject.search(regexp);
    返回第一次出现匹配指定正则表达式子串的下标，若没有匹配则返回 -1
    indexOf() 作用类似，但 indexOf() 不支持正则表达式

  • split()

    strObject.split(regexp,[howmany]);
    用一个指定的字符串或正则表达式，对原字符串进行拆分，返回拆得的子串数组
    若指定了 howmany 属性，则只返回拆得的前 howmany 个子串

正则表达式符号

元字符

  • 拥有特殊含义的字符

    元字符 描述
    . 匹配任意单个字符，换行符除外
    \w 匹配字母、数字、下划线（字符）
    \W 匹配非字母、数字、下划线（非字符）
    \d 匹配数字
    \D 匹配非数字
    \s 匹配空格字符
    \S 匹配非空格字符
    \n 匹配换行符
锚字符

  • 用于查找某个位置

    锚字符 描述
    ^ 匹配字符串的开头
    $ 匹配字符串的结尾
    \b 匹配字符串的边界
    \B 匹配字符串的非边界
    (?=exp) 匹配其后紧接exp的字符串
    (?!exp) 匹配其后没有紧接exp的字符串（exp代表任意表达式）
